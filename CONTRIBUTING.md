# Contribution Guide

Welcome Issues and PRs!

## Quick Claims

- [`good first issue`](https://github.com/CS-Faith/reasonix-migration-assistant/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22)
- [`help wanted`](https://github.com/CS-Faith/reasonix-migration-assistant/issues?q=is%3Aissue+is%3Aopen+label%3A%22help+wanted%22)

## Submitting Issues

- Bug: expected vs actual behavior + reproduction steps
- Feature Request: your scenario and need

## Submitting PRs

1. Fork -> Branch -> Commit -> PR
2. Describe the reason for the change
3. CI or manual test pass
